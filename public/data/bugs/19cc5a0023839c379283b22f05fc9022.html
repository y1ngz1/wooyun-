详细说明：

> server leiphone.com

Address:  113.106.88.130



> ls leiphone.com

[leiphone.com]

 leiphone.com.                  NS     server = ns3.leiphone.com

 leiphone.com.                  A      113.106.88.130

 a                              A      113.106.88.130

 ns2                            NS     server = s1.leiphone.com

 ns3                            NS     server = s2.leiphone.com

 s1                             A      112.95.235.130

 s2                             A      113.106.88.130

 s3                             A      113.106.88.150

 s4                             A      112.95.235.150

 s5                             A      113.106.88.144
